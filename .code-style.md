# Novel Bot Code Style

- PEP 8 compliant
- Type annotations on all public functions
- Google-style docstrings on public classes and functions
- from __future__ import annotations on all files
- Immutable data patterns (frozen=True dataclasses where appropriate)
- Max 50 lines per function, 400 lines per file
- No hardcoded values — use constants or config
- Explicit error handling, no silent swallowing
- Tests use AAA pattern (Arrange-Act-Assert)
